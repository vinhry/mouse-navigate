# Security

## Privacy model

MouseNavigate is a local macOS utility that:
- listens to global mouse side-button events,
- emits local keyboard/system actions,
- requires macOS Accessibility (and sometimes Input Monitoring) permissions.

MouseNavigate does not require network access to function.

## Reporting a vulnerability

Please report security issues privately to the maintainer before public disclosure.
Include:
- affected version
- reproduction steps
- expected vs actual behavior
- potential impact
